CONVENTIONS

What's new in v3 vs v2.

Key Changes from v2

→Amber replaces copper as the warm accent. Sourced from Edison string lights in photo 3, not a generic warm tone.
→Red is truer candy-apple (#B82018 vs #C0212B). Pulled directly from the Triumph tank.
→Dark mode base has blue tinge (#161A1E, not pure #111). Sourced from the dusk sky and roastery walls.
→Promo ticker is amber-dark (#A8620A), not red. Amber = warmth + promotion. Red stays for action only.
→Both modes in one file with a live toggle. CSS utility classes (text-heading, bg-surface, etc.) swap via body class.

Invariants (v1 → v3)

→Red = primary action. Teal = subscription/recurring. Amber = featured/highlight. These never swap.
→Bebas Neue / Boogaloo / Playfair / Barlow — same four fonts throughout all versions.
→rounded-sm everywhere except flavor tags (rounded-full) and modals (rounded-md).
→Roast intensity bar: teal (raw bean) → red (dark roast). Functional data encoded with brand color meaning.
→Flame stripe divider in header/footer. Thin. 3px. Gradient: dark red → candy → amber → back.
